Move between devices

Export on one device. Send the JSON file to yourself. On the new device, open this app and import the file. Your API key is never included in export.

Install on home screen

iPhone: Open in Safari → Share → Add to Home Screen.

Android: Open in Chrome → menu → Add to Home Screen.

Desktop: Use the install icon in the address bar if your browser shows it.

Privacy

Everything is stored locally on this device using localStorage. No account. No cloud sync. No hidden backend. The optional API key stays only in your browser storage and is never exported.
